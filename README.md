Developed a low-cost Braille learning device for visually impaired individuals, enabling typing on a serial monitor to generate tactile Braille output and translating input Braille into audible voice.

# Tech Used:
- Arduino UNO
- Push-Pull Solenoids
- HC-05
- Bluetooth module.
